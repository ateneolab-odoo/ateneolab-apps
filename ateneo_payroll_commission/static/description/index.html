Module to Manage Payroll Commissions

This module allows to generate commissions based on the amount (untaxed) invoiced.

Configure commissions

Set the commission percentage on the employee form view

If you check Works on Commissions? you must specify the percentage value for commission, a field for commission amount due in current period will be added to the employee contract form view.

Check your due commissions on contract

This field will take value every time a commission is generated for payroll and will be set to 0.00 once the payroll is Confirmed.

Set commissions on Payroll automatically

Every time you need to generate commissions to include them in your payroll, go to Payroll Application and create New Commission, set the period to take into account and press Generate. Automatically, this app will check all the invoices in this period for every user related to employees with valid contracts. Then one commission input will be generated for every employee.

After generating your current payroll, you will be able to view one input Commissions in every Payroll according to commission percentage established on employees form view. *** You will need account module in order to generate invoices***
